about AWS

리전 : 최소한 2개이상의 가용영역으로 구성됨 (이중화), 환경만 제공, 실제 구축은 customer

EC2 : 온프레미스관점에서 서버
      인스턴스 
     
EBS : 스토리지, 다운타임 없음, 필요할때마다 늘리면 됨
      block storage 
instance storage : 휘발성, but 고성능  
S3: 단독 스토리지, EC2와 별개로 존재할수 있음, 객체 스토어, 리전 레벨

인스턴스 :  중단후 재시작하면 호스트가 바뀔수 있음. EIP로 ip문제 해결
     
VPC : 프라이빗 가상 네트워크 Virtual Private Cloud
