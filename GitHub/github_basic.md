# GitHub Basic

 * ### git 구조
 ```
  working directory - staging area - local repository - remote repository
  <-------------------------- local -----------------> <------ remote ---->
                   git add        git commit        git push  
 ``` 

 * #### commit :

   local만 반영되고 remote repository는 아직 반영되지 않은 상태
   
 * #### publish :

   local(commit상태) -> remote repository로 반영
